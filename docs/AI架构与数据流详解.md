## 关键数据流

### 1. 聊天消息完整流程

```
┌─────────────────────────────────────────────────────────────┐
│  Step 1: 用户输入消息                                         │
└──────────────────────┬──────────────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  Flutter UI                                                  │
│  • 捕获用户输入: "帮我写一篇关于AI的文章"                      │
│  • 创建 StreamMessageParams {                                │
│      chat_id: uuid,                                          │
│      message: "帮我写...",                                   │
│      message_type: User,                                     │
│      question_stream_port: port1,                            │
│      answer_stream_port: port2,                              │
│    }                                                         │
└──────────────────────┬──────────────────────────────────────┘
                       │ FFI::AIEventStreamChatMessage()
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  Rust: AIManager::stream_chat_message()                     │
│  • 验证用户权限 (user_id, workspace_id)                      │
│  • 获取当前活跃模型: get_active_model(&chat_id)              │
│  • 获取或创建 Chat 实例: get_or_create_chat_instance()       │
└──────────────────────┬──────────────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  Chat::stream_chat_message()                                │
│  • 重置流控制标志: stop_stream = false                       │
│  • 创建 IsolateSink 连接 Dart                                │
│  • 调用 cloud_service.create_question()                     │
│    返回: ChatMessage {                                       │
│      message_id: 1234567890,                                │
│      content: "帮我写...",                                   │
│      author: Human,                                         │
│      created_at: 2025-01-08 10:00:00                        │
│    }                                                         │
│  • 发送问题ID到 question_stream: "message_id:1234567890"    │
│  • 保存问题到 SQLite                                         │
│  • 触发通知: DidReceiveChatMessage                           │
└──────────────────────┬──────────────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  Chat::stream_response() [async spawn]                      │
│  • 获取 workspace_id, ai_model                               │
│  • 调用 ChatServiceMiddleware::stream_answer()              │
└──────────────────────┬──────────────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  ChatServiceMiddleware 路由选择                              │
└──┬────────────────────────────────────────────────────────┬─┘
   │                                                        │
   │  云端AI                                                │  本地AI
   ▼                                                        ▼
┌──────────────────────────┐              ┌─────────────────────────┐
│ AppFlowy Cloud Service   │              │ LocalChatServiceImpl    │
│ • POST /chat/stream      │              │ • 检查 local_ai.is_ready│
│ • Authorization: Bearer  │              │ • 调用 local_ai         │
│ • SSE Stream             │              │   .stream_question()    │
└──────────┬───────────────┘              └────────┬────────────────┘
           │                                       │
           │                                       ▼
           │                              ┌─────────────────────────┐
           │                              │ LocalAI: LLM Chat       │
           │                              │ • 加载 RAG 上下文        │
           │                              │ • 调用 Ollama API        │
           │                              │ • 流式生成响应           │
           │                              └────────┬────────────────┘
           │                                       │
           └───────────────┬───────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  Stream<QuestionStreamValue>                                │
│  • Answer { value: "人工智能..." }                           │
│  • Answer { value: "是一门..." }                            │
│  • Metadata { sources: [...] }                              │
│  • FollowUp { should_generate: true }                       │
└──────────────────────┬──────────────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  Chat::stream_response() - 处理流                            │
│  while let Some(message) = stream.next() {                  │
│    match message {                                           │
│      Answer { value } => {                                   │
│        stream_buffer += value;                               │
│        answer_sink.send("data:" + value);  // 发送到 Dart    │
│      }                                                       │
│      Metadata { value } => {                                 │
│        answer_sink.send("metadata:" + json);                 │
│      }                                                       │
│      FollowUp { should_generate } => {                       │
│        answer_sink.send("follow_up:" + json);                │
│      }                                                       │
│    }                                                         │
│  }                                                           │
│  // 流结束                                                    │
│  let full_answer = stream_buffer.take();                     │
│  cloud_service.create_answer(full_answer, question_id);      │
│  answer_sink.send("done:");                                  │
└──────────────────────┬──────────────────────────────────────┘
                       │ IsolateSink
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  Dart: AppFlowyCompletionStream::_handleEvent()             │
│  • "data:人工智能" -> processMessage("人工智能")              │
│  • "data:是一门" -> processMessage("是一门")                 │
│  • "metadata:{...}" -> 解析并处理元数据                       │
│  • "done:" -> onEnd()                                        │
└──────────────────────┬──────────────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────────────┐
│  Flutter UI 更新                                             │
│  • 逐字显示AI响应                                             │
│  • 显示来源链接（如有）                                       │
│  • 显示相关问题建议（如有）                                   │
│  • 保存完整答案到本地                                         │
└─────────────────────────────────────────────────────────────┘
```

## 总结

### 关键设计原则

1. **分层架构**: UI、Service、Rust 三层清晰分离
2. **依赖注入**: 通过 Arc/Weak 管理依赖关系
3. **流式处理**: 使用异步流处理大模型响应
4. **状态管理**: 明确的状态机和状态转换
5. **错误处理**: 完善的错误传播和恢复机制
6. **扩展性**: 支持多种 AI 提供商和模型

### 性能优化

1. **并发处理**: Tokio 异步运行时
2. **缓存机制**: 模型配置、消息历史缓存
3. **向量索引**: 高效的相似度搜索
4. **流式响应**: 降低首字延迟
5. **资源管理**: 及时释放不用的资源

### 安全考虑

1. **API 密钥保护**: 环境变量、.gitignore
2. **输入验证**: 防止注入攻击
3. **权限检查**: 用户身份验证
4. **数据隔离**: Chat 之间数据隔离
5. **错误信息**: 不泄露敏感信息
